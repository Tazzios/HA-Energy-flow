Energy flow 

Proof of concept!

Wat doet het?

Het splits de energiestromen in huis op naar:
Zon → huis 
Zon → grid 
Zon → batterij
Grid → batterij
Batterij → huis
Batterij → grid

Per minuut wordt het aantal kWh bepaald. met de dynamic energycost integratie https://github.com/martinarva/dynamic_energy_cost kun je dit vervolgens omzetten in euro's. 

Waarom?
Bij het berekenen van de opbrengst in euro's is het een persoonlijke (of landelijke) kwestie hoeveel je rekent voor zon die direct verbruikt en zon die wordt teruggeleverd. 
D.m.v. deze integratie kun je daar nu verschillende prijzen aankoppelen



Code
- Code gaat er vanuit dat je AL je zon en batterij energy meters toevoegt, anders kun je vreemde resultaten krijgen.
- Zon wordt eerst in huis gebruikt.
